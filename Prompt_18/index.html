HTML (HyperText Markup Language) is the standard language used to create the structure of web pages. It defines the content and layout of a website.
CSS (Cascading Style Sheets) is used to style and format the layout of web pages. It defines how HTML elements should be displayed.
JavaScript is a programming language used to create dynamic content on web pages. It enables interaction and control of web elements.
Responsive design is an approach to web design that makes web pages render well on various devices and window sizes, such as desktops, tablets, and smartphones.
